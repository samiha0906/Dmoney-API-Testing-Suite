# Dmoney API Testing Suite

## Project Summary :
The Dmoney API Testing Suite is a framework designed to test the functionality and reliability of the Dmoney REST API using Postman. It ensures the smooth operation of user and transaction endpoints across various scenarios. This repository contains detailed test cases, Newman reports, API documentation, and a bug report to provide clear insights into the API’s performance.
## Technologies Used :
- Postman - For API request creation and execution.
- Newman - For running Postman collections via CLI and generating reports.
- Node.js - Dependency management for Newman.
## How to Run :
- Clone the Repository
- Install Dependencies with this command: “ npm install ‘’   
- Add .env file to the project root
-	Now hot this command to execute: “ npm test ”
-	After execution, the HTML report will be generated as report.html in the project directory. Open the file in any browser to view the test result.
